#Student Grade Tracker

## Description
A Java console application developed during the CodeAlpha Java Internship to manage student records and grades.

## Features
- Add Student
- Display All Students
- Search Student
- Edit Student Details
- Delete Student
- Grade Calculation (A, B, C, D, F)
- Pass/Fail Status
- Department and Section Management
- Summary Report

## Technologies Used
- Java
- OOP
- ArrayList
- Scanner
