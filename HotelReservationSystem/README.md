# Hotel Reservation System

## Description

The Hotel Reservation System is a Java console application developed during the CodeAlpha Java Programming Internship.

## Features

- View Available Rooms
- Search Rooms by Category
- Book Room
- Cancel Reservation
- Payment Simulation
- View Booking Details
- Standard, Deluxe and Suite Rooms

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- ArrayList
- Scanner
